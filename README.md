## Project title

Hotel room reservation system

## Project Overview

Hotel room reservation system that allows users to see available rooms, book and cancel room reservation, and request for refunds and cancellation of reservations according to hotel's policy. Supports VIP and standard room options, and optional accounts and email and PDF confirmations.

## Description

Backend built with PHP and MySQL. Communicates with the frontend via JSON for booking, payments, and confirmations.

## Features

- Book available rooms 
- Payment integration with cancellation and refund support  
- Email confirmations and PDF receipts  
- Optional account creation for loyalty and discount programs  

## Technology Used

- PHP  
- MySQL  
- Apache Server  
- API and JSON  

## Setup

Follow these instructions to setup the project locally on your machine.

### Prerequisites
Ensure you have installed PHP and php package manager composer on your local machine.
 
1. Clone this repository:

https://github.com/five-code-nerds/Hotel_Room_Reservation-Backend.git

2. Install necessary packages:

run this command ```composer init```

3. Run the local server

run this command ```php -S localhost:3000 -t public```
